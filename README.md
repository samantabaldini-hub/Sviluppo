# Simulazione di un processo produttivo in Python

Questo progetto nasce come parte di un project work universitario dedicato alla simulazione di un processo produttivo nel settore secondario.  
L’obiettivo è rappresentare, in modo semplificato ma realistico, le dinamiche di una linea di produzione mediante codice Python, con generazione di dati casuali e creazione di un report finale in formato CSV.

## ⚙️ Funzionalità principali
- Generazione casuale dei prodotti con parametri operativi differenti  
- Calcolo dei tempi di produzione giornalieri e totali  
- Creazione automatica di un report finale in formato CSV  
- Output immediato su console e salvataggio dei risultati  

## 🧠 Strumenti utilizzati
- **Python** come linguaggio di programmazione principale  
- **pandas** per la gestione dei dati tabellari  
- **random** e **datetime** per la simulazione e la generazione dei report
